# Terminology

+ Issuer (iss) - identifies principal that issued the JWT
+ Subject (sub) - identifies the subject of the JWT
+ Audience (aud)
+ Expiration time (exp)
+ Not before (nbf)
+ Issued at (iat)
+ JWT ID (jti)
+ Token type (typ) - If present, it is recommended to set this to JWT
+ Content type (cty)
+ Message authentication code algorithm (alg)
